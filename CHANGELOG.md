- Router + Answerer agent with DeepSeek LLM
- Vector search over mod descriptions (Qdrant + BGE-M3)
- Mod recommendation by genre/tags (LLM auto-tagging pipeline)
- Mod compatibility lookup (known_integrations extraction)
- Multi-turn conversation memory
- Critic agent with retry loop for answer quality
- ChatGPT-style Gradio UI with debug panel
- MySQL storage for mod metadata, tags, and integrations
